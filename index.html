About This Project

This website is a simple static web application hosted using Google Cloud Storage. It demonstrates how content such as HTML, CSS, and JavaScript can be deployed on a cloud platform without requiring a backend server.

Cloud Services Used

  • Cloud Storage - for hosting files
  • IAM (Identity and Access Management) - for public access control
  • Bucket configuration - for website hosting

Architecture

User requests the website through a browser. The request is served directly from a Cloud Storage bucket configured for static hosting. Since the content is static, no server-side processing is required.

This ensures faster performance, low cost, and high scalability.
